# TravelNest AI

TravelNest AI is a smart travel companion app designed to make journeys simple, safe, and connected.  
It helps travelers with offline messaging, emergency signaling, and AI-powered trip planning.

## Features
- 🌍 Offline-first travel assistant
- 💬 Messaging even in low-connectivity areas
- 🚨 Emergency signaling support
- 🎵 Media playback during travel
- 💳 Future-ready offline payments

## Tech Stack
- HTML, CSS, JavaScript
- Supabase (database + auth)
- Vercel (hosting)
- Anthropic AI (intelligent assistant)
